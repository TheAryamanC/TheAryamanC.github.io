This is an interactive tool for exploring Boolean satisfiability (SAT) problems. You can define propositions and constraints, then watch a solver search for a truth assignment that satisfies them all.

It runs in Unity and includes a built-in stochastic local search solver, DIMACS encoding for handing problems off to external SAT solvers, and a solution enumerator for browsing the different ways a problem can be satisfied.

You can find the source code here.

All credit to Ian (as usual) for creating the original base project, upon which I added complete SAT solvers and #SAT extensions.

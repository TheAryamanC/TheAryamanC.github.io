This is a simple text-based parser for OpenStreetMap data. It parses the XML format used by OpenStreetMap and allows the user to query the data (e.g. find buildings, directions between locations etc.).

Honestly, there's not much to say about this project except that it was the first long-term project I worked on.

You can find the code here, but it's not very well maintained.
